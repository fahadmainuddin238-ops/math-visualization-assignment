# Math Visualization Assignment

## Project Description

This project demonstrates mathematical function visualization and student score data analysis using Python. Different mathematical graphs and data visualization techniques were implemented using NumPy and Matplotlib.

---

## Libraries Used

- NumPy
- Matplotlib

---

## How to Run the Code

Install required libraries:

```bash
pip install numpy matplotlib
```

Run the Python files one by one:

```bash
python task1_math_visualization.py
python task2_own_eq.py
python task3_score_visualization.py
python task4_prediction.py
```

## Short Explanation

### How does visualization help us understand mathematical functions and data?

Visualization helps us easily understand patterns, trends, relationships, and behaviors of mathematical functions and datasets. Graphs make complex information easier to analyze visually.

---

### Which plot was most useful in this assignment and why?

The best-fit prediction graph was the most useful because it clearly showed the relationship between midterm and final scores and demonstrated simple prediction using data.

---

### What is the role of NumPy and Matplotlib in your project?

NumPy was used for mathematical calculations, arrays, and generating data values. Matplotlib was used to create graphs, charts, labels, legends, and visualizations.
